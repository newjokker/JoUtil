# 类的使用手册


### 命名规则

* add       增加
* parse     解析
* count     计数
* crop      裁剪
* del       删除
* do        执行
    * offset    偏移
    * do nms    
* draw      画图
* filter    筛选
* get       获取
* print     打印
* save      保存


### 主要使用的几个文件

* resBase         
    * deteObj           矩形框
    * deteAngleObj      有角度的矩形框
* deteXml               xml 解析
* deteRes               xml 信息的保存和操作类
* resTools              辅助函数
