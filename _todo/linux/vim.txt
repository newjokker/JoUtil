


* vim
    * vim +数字 文件路径，光标移动到指定行
    * vim +/关键词 文件路径 ，打开指定文件，高亮显示关键词，好像 ubantu 上不好高亮
    * vim 文件路径1 文件路径2 文件路径3  ，同时打开多个文件

* vim 配置
    * 临时: 在文件打开的时候，末行模式下面显示的文件
    * 个人: 在家目录下面，~/.vimrc ，没有的话需要自己创建
        * set nu , set nonu 显示行号， 不显示行号
        * syntax off ，关闭色彩显示
        * set paste，拷贝代码的时候，不会出现很多的空格
    * 全局: /etc/vimrc
    * 配置文件优先级，个人 > 全局

* vim 异常退出，突然断电等情况导致的
    * 发现交换文件 .swp ，编辑过程中产生的临时文件，原始的文件名前面加 . 后面加 .swp

* 别名机制，创建一些自定义的命令
    * alias ll='ls -l'
    * 使用别名映射文件，~/.bashrc 中，改完之后需要重新登录当前用户


* 退出方式
    * :q , :wq
    * :x 如果编辑了就 :wq 如果没有编辑就是 :q
